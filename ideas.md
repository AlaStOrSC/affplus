<idea>
  **Design Movement**: Cyberpunk Industrial
  **Core Principles**:
  1. **High-Contrast Impact**: Fondos oscuros profundos con acentos de neón vibrantes para guiar la atención.
  2. **Tech-Forward Aesthetics**: Uso de elementos de interfaz de usuario inspirados en HUDs, líneas de cuadrícula sutiles y tipografía monoespaciada para detalles técnicos.
  3. **Dynamic Motion**: Animaciones de desplazamiento suaves y efectos de brillo que reaccionan a la interacción del usuario.
  4. **Data Visualization**: Representación abstracta de flujos de datos y redes para simbolizar la infraestructura de tráfico.

  **Color Philosophy**:
  - **Fondo**: Negro profundo (#050505) y gris carbón (#121212) para crear profundidad y contraste.
  - **Acentos**: Verde neón (#39FF14) para acciones positivas y crecimiento (tráfico, ROI). Púrpura eléctrico (#BC13FE) para tecnología y sofisticación.
  - **Texto**: Blanco (#FFFFFF) para legibilidad máxima sobre fondos oscuros, con gris claro (#A0A0A0) para texto secundario.
  - *Intención*: Evocar una sensación de tecnología avanzada, velocidad y rendimiento nocturno, típico del mundo del iGaming y la tecnología de alto nivel.

  **Layout Paradigm**:
  - **Asymmetric Grid**: Romper la rigidez de las cuadrículas tradicionales con elementos superpuestos y alineaciones descentradas.
  - **Section Flow**: Transiciones fluidas entre secciones sin bordes duros, utilizando degradados y efectos de paralaje.
  - **Sticky Navigation**: Menú lateral o flotante que acompaña al usuario, con indicadores de progreso de lectura.

  **Signature Elements**:
  - **Glowing Borders**: Bordes de tarjetas y botones con efectos de brillo sutil (box-shadow) en los colores de acento.
  - **Abstract Data Streams**: Gráficos de fondo animados que sugieren movimiento de datos y conexiones de red.
  - **Glassmorphism Dark**: Paneles semitransparentes con desenfoque de fondo para separar el contenido del fondo dinámico.

  **Interaction Philosophy**:
  - **Hover Effects**: Elementos que se iluminan o cambian de color al pasar el cursor, simulando activación tecnológica.
  - **Scroll-Triggered Animations**: Elementos que aparecen o se ensamblan a medida que el usuario se desplaza hacia abajo.
  - **Micro-interactions**: Feedback visual inmediato en botones y enlaces.

  **Animation**:
  - **Entrance**: Fade-in con desplazamiento vertical suave para textos e imágenes.
  - **Continuous**: Pulsaciones suaves en elementos clave (CTAs) y movimiento lento en fondos abstractos.

  **Typography System**:
  - **Headings**: 'Space Grotesk' o 'Orbitron' para títulos, transmitiendo modernidad y tecnología.
  - **Body**: 'Inter' o 'Roboto' para legibilidad en bloques de texto largos.
  - **Technical Details**: 'JetBrains Mono' o 'Fira Code' para datos, estadísticas y etiquetas pequeñas.
</idea>

<idea>
  **Design Movement**: Fluid Glassmorphism
  **Core Principles**:
  1. **Depth & Layering**: Uso extensivo de capas translúcidas para crear una sensación de profundidad inmersiva.
  2. **Organic Shapes**: Formas fluidas y curvas en los fondos para contrastar con la estructura del contenido.
  3. **Light & Shadow**: Juego sofisticado de luces y sombras para definir volúmenes y jerarquías.
  4. **Vibrant Gradients**: Degradados complejos y ricos que mezclan los colores de la marca de forma suave.

  **Color Philosophy**:
  - **Fondo**: Degradados oscuros profundos que van del azul marino al púrpura (#0F172A a #3B0764).
  - **Acentos**: Verde esmeralda brillante (#10B981) y orquídea radiante (#D946EF) aplicados en degradados y brillos.
  - **Vidrio**: Blanco semitransparente con desenfoque de fondo (backdrop-filter).
  - *Intención*: Crear una experiencia premium y etérea que sugiera sofisticación tecnológica y fluidez de datos.

  **Layout Paradigm**:
  - **Floating Cards**: El contenido reside en "tarjetas de vidrio" que flotan sobre el fondo fluido.
  - **Central Axis**: Alineación central para la narrativa principal, con elementos flotantes alrededor.
  - **Immersive Scroll**: El fondo evoluciona y cambia suavemente a medida que el usuario se desplaza.

  **Signature Elements**:
  - **Frosted Glass**: Paneles con efecto de vidrio esmerilado para contenedores de texto.
  - **Glowing Orbs**: Orbes de luz difusa que se mueven lentamente en el fondo.
  - **Thin Borders**: Bordes de 1px con degradados sutiles para definir límites sin ser agresivos.

  **Interaction Philosophy**:
  - **Parallax**: Movimiento de capas a diferentes velocidades para acentuar la profundidad.
  - **Tilt Effects**: Tarjetas que se inclinan sutilmente en 3D al mover el ratón sobre ellas.

  **Animation**:
  - **Liquid Motion**: Transiciones fluidas y orgánicas entre estados.
  - **Blur Transitions**: Elementos que se enfocan y desenfocan al entrar y salir de la vista.

  **Typography System**:
  - **Headings**: 'Outfit' o 'Plus Jakarta Sans' para un look moderno y geométrico pero amigable.
  - **Body**: 'Inter' o 'Satoshi' para una legibilidad limpia y neutral.
</idea>
